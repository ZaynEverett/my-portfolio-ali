# Ali Ahmed - Premium Portfolio Website

A stunning, futuristic personal portfolio website built with React, Tailwind CSS, and Framer Motion. Features advanced animations, glassmorphism effects, interactive components, and a modern dark theme with neon accents.

## Features

✨ **Advanced Animations & Effects**
- Typing animation in hero section
- Scroll reveal animations (fade, slide, zoom)
- Parallax scrolling effects
- Smooth hover animations on all elements
- Particle background animation

🎨 **Modern Design**
- Dark mode with futuristic neon glow UI
- Glassmorphism effects
- Gradient backgrounds
- Fully responsive design
- Beautiful micro-interactions

🎯 **Key Sections**
- **Hero**: Engaging landing section with typing animation and CTA buttons
- **Skills**: Advanced animated progress bars and circular skill indicators
- **Projects**: Dynamic project cards with live demo and GitHub links
- **Services**: Modern service cards with animations
- **Experience**: Professional timeline with enhanced animations
- **Education**: Card-based education section with certifications
- **Contact**: Modern contact form with validation and social links
- **ChatBot**: Floating AI chatbot interface for visitor engagement

🚀 **Advanced Features**
- Custom animated cursor
- Sticky navbar with glass effect
- Active section highlight on scroll
- Back-to-top button with smooth scrolling
- Animated page loader
- Particle background effects
- Fully accessible components

## Tech Stack

- **Frontend Framework**: React 18
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Language**: TypeScript
- **Icons**: React Icons

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd "Ali Ahmed/Portfolio ali ahmed"
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will open in your browser at `http://localhost:3000`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally

## Customization

### Personal Information

Update the following files with your information:
- `src/components/Hero.tsx` - Name, tagline, and description
- `src/components/Contact.tsx` - Email, phone, and social links
- `src/components/Footer.tsx` - Contact information

### Colors

Modify the Tailwind color configuration in `tailwind.config.js`:
- `neon-pink` - Primary accent (#ff006e)
- `neon-blue` - Secondary accent (#00d9ff)
- `neon-purple` - Tertiary accent (#9d00ff)
- `dark-bg` - Background color (#0a0e27)

### Skills & Projects

Update the skills and projects arrays in their respective component files to showcase your work.
